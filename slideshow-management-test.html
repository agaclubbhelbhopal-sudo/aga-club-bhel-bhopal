🚀 How to Use

Quick Start Guide:
  1. Access Admin Panel: Navigate to /admin-panel.html
  2. Select Page: Choose the page from the dropdown menu
  3. View Slideshow: See current slideshow images in the management grid
  4. Add Images: Click "Add New Image" and upload files
  5. Reorder Images: Drag and drop to rearrange the order
  6. Preview Changes: Use the real-time preview to see results
  7. Save Changes: All changes are saved automatically
