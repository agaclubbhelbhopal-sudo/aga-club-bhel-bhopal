Filename Extractor

Instructions

  • Either paste a list of file paths in the input area OR select files using the file selector
  • Click "Extract Filenames" to get just the filenames with extensions
  • Click "Copy to Clipboard" to copy the results
  • Use "Add Prefix/Suffix" to add text before/after each filename
  • Use "Join with Comma" to create a comma-separated list
